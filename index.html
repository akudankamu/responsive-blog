So what's this all about then?

Posted on November 1, 2012 by admin

Why hello there, I’m creating or created (hmm, I’m already having a tense problem). a blog website. The purpose of this project is to create a responsive website that uses the current best practices and tools within front end web development right now (Nov, 2012 ). Let me explain, see I really want to be a front end web developer, for the last couple of years my work has more been in support, so I spent all day debugging customers websites ( WP e-Commerce sites, so WordPress e-Commerce ), but enough of that. I want to build again. While I have sort of keep up with the developments in web technology. The latest techniques on the front end have taken a massive step forward since I last built a website. Key emoung this is the mobile and device revolution. Also the latest open standards are pretty much safe to use now ( like HTML5/CSS3). There has also been massive leaps in writing css, that is, now preprocessors are all the rage (less,sass). I haven’t used these before but I can’t wait. It looks amazing. Also javascript now demands deeper knowledge for serious front end devs. This is all the tech side but I am also equally keen on exploring the UI design side of things. It seems there have been massive progress in UI theory in the web. Once designing a site was similar to designing print, as in you could create on mockup in photoshop then code that up to get it as close as possible in a small handful of browsers on the desktop. There was a very pagy methodology  to it all. These days however sites are not a collection of pages ( or they do not have to be ). It more about presenting the data that the user wants. Hierarchies of data if you will, with effortless access to content. I’ll talk more about this in a later post.

So the point is I will be creating this little site as a project to learning about all these things. I will be writing about my learning, challenges, and insights I discover while creating this site. I hope to learn a lot, and I hope my rants are interesting to you the reader..err user/potential client/employer!
